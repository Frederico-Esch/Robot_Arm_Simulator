# Dependencias para compilação

- CMAKE
- RAYLIB
- C++ Compiler

# Compilação

```bash

cmake -B build
cd build
make; ./braco

````

# Comandos

- P: Tocar animação
- Setas Direita/Esquerda: Aumenta e diminui ângulo da junta
- Setas Cima/Baixo: Troca junta
- Z: Foca na origem
- X: Zera ângulos
- M: Alterna entre origem global e origem da junta em foco
- +/-: Aumentar e diminuir velocidade de movimentação
- A/W/S/D/SPC/CTRL: Movimentaçâo
- Mouse: Câmera
